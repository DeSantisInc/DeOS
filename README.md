## Getting Started

### Install

#### *1. Clone the Repo*

```sh
Δ git clone git@github.com:DeSantisInc/DeOS.git && cd DeOS
```

#### *2. Bootstrap the Environment*

```sh
Δ sh bootstrap.sh
```
